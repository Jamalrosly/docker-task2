## Project Description
This project demonstrates containerization using Docker and deployment on AWS EC2.  
A simple website displaying my basic details is deployed using Dockerfile and docker-compose.

## Tech Stack Used
- AWS EC2
- Docker
- Docker Compose
- Nginx

## Project Steps
1. Launched AWS EC2 instance.
2. Installed Docker and Docker Compose.
3. Created HTML file with personal details.
4. Created Dockerfile using Nginx image.
5. Created docker-compose.yml file.
6. Built and deployed container.
7. Accessed website using EC2 public IP.

## Output
The website successfully displays personal details through Docker container running on EC2.
